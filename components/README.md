# Components

- `sidebar-layout.tsx`: Sticky docs-style shell with collapsible navigation and system status block.
- `command-palette.tsx`: `Ctrl/Cmd+K` search and quick actions.
- `recruiter-mode-toggle.tsx`: Global recruiter vs detailed view toggle.
- `project-card.tsx`: Reusable project preview card for overview/projects listing.
- `projects-explorer.tsx`: Search + tag filters + keyboard `/` focus integration.
- `project-case-study.tsx`: Recruiter summary + deep technical sections for project detail pages.
- `system-overview.tsx`: Project-specific architecture visual plus data flow bullets.
- `metrics-panel.tsx`: Results/evidence list panel.
- `key-decisions-table.tsx`: Tradeoff/rationale table.
- `callout.tsx`: Inline emphasis block for MDX notes.
- `code-block.tsx`: Inverted code block rendering.
- `copy-to-clipboard-button.tsx`: Reusable clipboard action.
- `section-divider.tsx`: Lightweight 1px section separator.
- `timeline.tsx`: Experience timeline component.
- `contact-panel.tsx`: Direct contact hub for email, resume, and profile links.
